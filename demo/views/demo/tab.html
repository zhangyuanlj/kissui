JS代码
//添加TAB
$("#addTab").bind("click", function(){
    if($(".tab:eq(2) li").length < 4){
        tab3.addTab({
            tabTitle : $(".tab:eq(2) li").eq(0).clone(),
            tabCon : $(".tab_content:eq(2) div").eq(0).clone()
        });
    }
});
//删除TAB
$("#deleteTab").bind("click", function(){
    if($(".tab:eq(2) li").length > 2){
        tab3.deleteTab($(".tab:eq(2) li").length-1);
    }
});
配置说明
属性 说明 是否必填
$tabTitle 标签标题 必填
$tabCon 标签内容 必填
selectIndex 选中TAB的索引 可选
selectStyle 选中TAB的样式名称 可选
triggerEvent 触发切换的事件（click或者mouseover） 可选
slideEnabled 是否自动切换 可选
slideInterval 切换时间间隔 可选
slideDelay 下一次切换间隔 可选
autoInit 是否自动初始化 可选
onShow(KTab, $selectTab, $selectCon) 每次切换完成后的回到函数 可选
K.Ui.Tab方法
方法名 作用
init 初始化Tab
selectTab(selectIndex) 选中标签
addTab(tab) 添加标签
deleteTab(selectIndex) 删除标签
