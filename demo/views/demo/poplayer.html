遮罩层JS代码
//动态创建遮罩层，需要导入maskLayer代码包
$("#showMask").bind("click", function(){
    var mask = new KMask({
        onMaskLayerClick : function(KMask){
            KMask.fadeOut(300, function(){
                KMask.remove();
            });
        }
    });
    mask.fadeIn(500, function(){
        mask.setOpacity(0.5);
    });
});
HTML属性配置说明
属性 说明 是否必填
type 指定该控件要显示dropdown或者poplayer 必填（值为ks-dropdown）
data-target-obj 显示dropdown或者poplayer的容器ID 必填
data-layer dropdown或者poplayer的ID 必填
data-width dropdown或者poplayer的宽度 可选（默认值为240）
data-location dropdown或者poplayer的位置 可选
data-show 是否默认就显示 可选
data-trigger-mode 显示dropdown或者poplayer的事件名称 可选（值为mouseover或者mousedown）

KDropDown配置说明
属性 说明 是否必填
$targetObj 显示dropdown或者poplayer的容器ID 必填
$layer dropdown或者poplayer的ID 必填
layerWidth dropdown或者poplayer宽度 可选，默认值为240
location dropdown或者poplayer显示的位置 可选
show 是否默认就显示 可选
triggerMode 显示dropdown或者poplayer的事件名称 可选（值为mouseover或者mousedown）
KDropDown方法
方法名 作用
layerShow 显示dropdown或者poplayer
layerHide 隐藏dropdown或者poplayer
