<div class="ks-button-wrap ks-button-group">

    	<button class="ks-button-input ks-button-primary" id="tooltip2" type="ks-tooltip" data-event-obj="#tooltip2" data-width="240" data-content="这里是对话框的大段文字这里是对话框的大段文字这里是对话框的大段文字" data-location="top" type="button">Button</button>
        <button class="ks-button-input ks-button-group-last ks-button-primary" id="tooltip3" type="ks-tooltip" data-event-obj="#tooltip3" data-width="240" data-content="这里是对话框的大段文字这里是对话框的大段文字这里是对话框的大段文字" data-location="left" type="button">Button</button>
</div>
HTML属性配置说明
属性 说明 是否必填
type 指定该控件要显示tooltip 必填（值为ks-tooltip）
data-event-obj 显示tooltip的容器ID 必填
data-width tooltip宽度 可选，默认值为200
data-content tooltip内容 可选
data-location tooltip显示的位置 可选
data-dynamic-enable 是否动态计算tooltip的位置 可选
data-dynamic-direction 是在水平方向（1）上计算，还是在垂直方向（2）上计算 可选
data-animate 是否以动画方式显示tooltip 可选
data-offset tooltip偏移的位置 可选

代码创建
实例
代码创建
JS代码
/*需要导入position、tootip代码包*/
var tooltip = new KTootip({
    $eventObj : $("#tooltip4"),
    width : 120,
    dynamic : {
        enable : true,
        direction : 2
    }
});
KTootip配置说明
属性 说明 是否必填
$eventObj 显示tooltip的容器ID 必填
width tooltip宽度 可选，默认值为200
content tooltip内容 可选
location tooltip显示的位置 可选
dynamic.enable 是否动态计算tooltip的位置 可选
dynamic.direction 是在水平方向（1）上计算，还是在垂直方向（2）上计算 可选
animate 是否以动画方式显示tooltip 可选
offset tooltip偏移的位置 可选
KTootip方法
方法名 作用
gettooltipId 获取创建tooltip的ID
destroy 销毁创建的tooltip
