创建对话框
实例
内容对话框 警告对话框 确认对话框
代码
/*需要导入drag、dilog代码包*/
//内容对话框
var dilog = new K.Ui.Dilog({
    width : 300,
    title :　{
        text : "选择人员"
    },
    content : {
        text : "创建了一个内容对话框！！！呵呵！！！"
    },
    btn : {
        show : true,
        align : "center",
        items : [{
            text : "确定",
            btnStyle : "blue",
            callback : function(Dilog){
                Dilog.setPosition(8, {left:10, top:20});
                return true;
            }
        },
        {
            text : "更新内容",
            btnStyle : "gray",
            callback : function(Dilog){
                Dilog.setContent({
                    loadType : 1,
                    text : "对话框内容改变了，哈哈！！！"
                }, 5);
                return true;
            }
        },
        {
            text : "取消",
            btnStyle : "gray",
            callback : function(Dilog){
                Dilog.destroy();
                return true;
            }
        }]
    }
});
//警告对话框
var alertDilog = new K.Ui.Dilog({
    isConDilog : false
});
alertDilog.createAlert(280, "警告对话框", "警告对话框内容");
//内容对话框
var confirmDilog = new K.Ui.Dilog({
    isConDilog : false
});
confirmDilog.createConfirm(440, "确认对话框", "确认对话框内容确认对话框内容确认对话框内容确认对话框内容确认对话框内容", function(Dilog, $btn){
    Dilog.destroy();
});
var confirmDilogId = confirmDilog.getDilogId();
//注册拖动
var drag = new K.Ui.Drag({
    hander : $("#"+confirmDilogId+" .ks-dilog-title"),
    dragObj : $("#"+confirmDilogId)
});
drag.init();
配置说明
属性 说明 是否必填
$parent 插入对话框的节点 可选（默认为当前页面的body节点）
width 对话框宽度 可选
position 对话框位置 可选，默认垂直居中
styleName 对话框样式名称，可以更改对话框外观 可选
isConDilog 是否为内容对话框 可选（创建警告和确认对话框时要设为false）
title.show 是否显示标题 可选
title.text 标题文本 可选
title.text 标题文本 可选
content.loadType 内容载入方式 可选（1：普通；2：异步;）
content.text 内容文本 可选
content.loadUrl 异步加载时的页面地址 可选
content.modal 是否为模态窗口 可选
content.btn.show 是否显示按钮 可选
content.btn.align 按钮对齐方式 可选（left、center、right）
content.btn.items 按钮组 可选
K.Ui.Dilog方法
方法名 作用
getDilogId 获取对话框ID
setPosition(integer position, integer value) 设置对话框位置
setContent(string content, integer position) 设置对话框内容
createAlert(integer width, string title, string content) 创建警告对话框
createConfirm(integer width, string title, string content, function callback) 创建确认对话框
destroy 销毁对话框
